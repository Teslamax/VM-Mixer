# 🎛️ ANAVI Knobs 3

**ANAVI Knobs 3** is the primary long-term distributed mixer control surface for VM-Mixer.

---

## 🎹 Planned Interaction Model

- `rotate` = gain adjustment
- `press` = mute toggle
- `double_press` = reset to 0 dB
- `hold` = remote restart action

---

## 🔌 Preferred Transport

- USB CDC serial with semantic JSON events
